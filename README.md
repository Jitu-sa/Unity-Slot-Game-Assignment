# \# Unity Slot Machine Game

# 

# \## Game Overview

# This is a simple slot machine game developed in Unity.

# The player selects a bet amount and spins the reels to try to match symbols and win rewards.

# 

# \## Features

# \- Smooth reel animations

# \- Randomized symbol generation

# \- Reward system based on bet amount

# \- Clean UI system

# 

# \## How to Run the WebGL Build

# 1\. Open the `Build/WebGL` folder

# 2\. Run `index.html` in a browser

# OR

# Upload the build to itch.io or a web server.

# 

# \## Bonus Features

# \- Adjustable bet system

# \- Reward multiplier

# 

# \## My Approach

# I built the slot machine using a reel-based animation system.

# Random symbols are generated using Unity's random function.

# A dropdown system allows the user to change betting amounts and rewards are calculated accordingly.
